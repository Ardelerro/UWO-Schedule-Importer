### To use:

1. **Download the ZIP file**
   - Go to the **Releases** section of this repository
   - Download the attached `.zip` file for this release

2. **Extract the ZIP**
   - Right-click the ZIP file and extract it to a folder on your computer
   - Ensure the extracted folder contains files such as `manifest.json`, `popup.html`, etc.

3. **Open Extensions**
   - Open Google Chrome or another chromium based browser
   - Navigate to:
     ```
     chrome://extensions
     ```
     Or to your extensions page

4. **Enable Developer Mode**
   - Toggle **Developer mode** ON (top-right corner)

5. **Load the extension**
   - Click **Load unpacked**
   - Select the extracted folder (not the ZIP file)

6. **Verify installation**
   - The Schedule Importer extension should now appear in your extensions list
   - Optionally pin it to the toolbar using the puzzle icon

7. **Use the extension**
   - Click the extension icon to open the popup
   - Navigate to your timetable on draft my schedule
   - Follow the on-screen instructions to import your schedule

### Notes
- This extension is currently distributed as an unpacked extension.
- No user data is sold or shared with third parties.
